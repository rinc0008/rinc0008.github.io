Grid

Bootstrap have a predeterminate grid system of 12 columns made with flexbox! To organize the content in the grid we must first have the content wrapped in a 'div' with the class ‘container’ this will allows us to select which part of the content we want to reorganize in the grid. It will be responsive automatically, because the system will identify which screen size are you using for designing the grid! How so? When giving the command to organize the element in the columns with the class ‘.col-lg-*’ (*= the column number, if it is 5 it will take 5 columns of 12) Bootstrap will identify that is for long screens (≥992px).

If designing for mobile, below 576px the class will be ‘.col-xs-*’, for tablets (≥576px) ‘.col-sm-*’, for medium screens (≥768px) ‘.col-md-*’, for more longer screens (≥1200px) ‘.col-xl-*.

When we’re referring to the system grid the standard of 12 columns would work better for the desktop, tablet will use 8 and mobile will use 6 columns. Nevertheless, it can be changed and/or reversed! If in the div container it is specified, the number of columns for each screen size. For example, if using in the container ‘div class="col-xs-12 col-sm-6 col-md-8”’ it will set the mobile screen to have 12 columns, the tablet 6 columns and the desktop 8 columns.
