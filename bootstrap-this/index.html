How to start with Bootstrap?

First, we need to link the HTML5 project with Bootstrap, to do so go to getboostrap.com>Introduction and there under CSS title you will find a 'link' code which we will need to copy and paste in the 'head' of our HTML5 file. As simple as that, then you are ready to start using the components.

Bootstrap is about mobile first design and structures, the use of media queries is the key to responsive layouts specially when using the powerful grid templates and in general to scale up elements and optimize in all screen sizes. To warrant its function we need to add the ‘responsive viewport meta tag’ to the HTML5 'head;. It can be found at getboostrap.com>Introduction and there under responsive meta tag.

Is there another way to start?

Yes! The official website getboostrap.com also provide us with a starter template which already includes all the information above! Links to the Css and the meta tag for mobile first (media queries) and the Javascript plugins is all provided in a HTML5 structure! Ready to copy-paste, edit, and add components on it. Keep in mind that Bootstrap Version 4.4 (the one we are working with) is compatible with HTML5 to display correctly, with other html version the designs can show incomplete.

There are 3 more ways to start with Bootstrap which requires download files as the CSS and JS precomposed, Source files and Package managers that are mostly used to work in a local directory, for more complex projects or for development purpose.

Basic styles?

Bootstrap uses a ‘Reboot’ which is like a kit of code reset that allows the style to be consistent across the browsers or platforms (rewriting the default style of the browsers). So that way it can display its styles as planned. The nicest thing about Bootstrap is that is built with minimal inherit styles which means that it’s easier to modify by increasing the specificity when selecting an element (wrapping the content in a "div" is a good way to do it). Some of the styles include:

  • Images
  • List
  • Headings
  • Font’s utilities
  • Colour’s utilities
  • Blockquotes
